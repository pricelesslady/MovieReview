## ⚙️ How It Works
1. User enters a movie name in the search bar  
2. The app fetches data from the OMDb API  
3. Movies are displayed dynamically in a grid  
4. Clicking a movie shows detailed information  
5. Users can:
   - Vote for actors  
   - Submit their favorite scenes
