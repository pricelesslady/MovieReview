# 🎬 CineVerse – Movie Review Web App

## 📌 Project Description
CineVerse is a dynamic movie web application that allows users to search for movies, view detailed information, vote for their favorite actors, and share memorable scenes.

The project integrates an external API to fetch real-time movie data and provides interactive features to enhance user engagement.

---

## 🚀 Features
- 🔍 Search for movies using the OMDb API  
- 🎥 View movie details (poster, plot, actors, rating)  
- 🗳 Vote for your favorite actor in a movie  
- 🎬 Submit and display favorite scenes  
- 📱 Fully responsive design for mobile and desktop  
- 🎨 Modern UI with animations and hover effects  

---

## 🛠 Technologies Used
- **HTML5** – Structure of the web application  
- **CSS3** – Styling, layout, and responsiveness  
- **JavaScript (ES6)** – Interactivity and logic  
- **OMDb API** – Fetching movie data
